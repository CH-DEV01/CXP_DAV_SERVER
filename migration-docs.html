¿Por qué Spring Boot 3.3.0?
Spring Boot 3.x es el primer release que usa el namespace jakarta.* (requerido por Tomcat 10.1+ / Tomcat 11). La versión 3.3.0 es la mínima que incluye soporte estable para Tomcat 11 como servidor externo (Servlet API 6.1 / Jakarta EE 11).

PASO 3 Migración javax.* → jakarta.* — 27 clases ✓ Aplicado

Spring Boot 3 abandona el namespace javax.* en favor de jakarta.* (Jakarta EE 9+). Tomcat 11 implementa Servlet 6.1 (Jakarta EE 11) y no reconoce las clases javax.servlet.* — sin este cambio el proyecto no compila. Se migraron 27 clases en un único paso mediante reemplazo masivo de imports.
